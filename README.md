# day-7

# Why Testing Matters
Testing helps find bugs before users face problems. It ensures the system works correctly and securely after every change.

# What is Asynchronous Apex?
Asynchronous Apex runs processes in the background instead of immediately. It is used for large data processing, scheduled jobs, and callouts

# What is Salesforce DX?
Salesforce DX is a development tool that helps teams manage coding, testing, version control, and deployment efficiently.

# Complete System Workflow (End-to-end explanation)
User enters data → validation happens → automation/triggers execute → data is stored → reports and notifications are generated for users.

#  Important Test Cases (Your examples)
Check if records save correctly, validate mandatory fields, test trigger automation, verify user permissions, and ensure reports display correct data.

# Reflection: Why enterprise software development needs structured workflows
Structured workflows improve accuracy, teamwork, and maintenance. They help large organizations manage complex processes in a reliable way.
